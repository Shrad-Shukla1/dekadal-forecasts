About

These forecasts show bias-corrected, downscaled dekadal (10-day) precipitation predictions for Africa from the SubC project (https://weather.ou.edu/~kpegion/subc/).

Data Source

CHC Data Repository

Models

  • NCEP CFSv2 - Climate Forecast System Version 2
  • GMAO GEOS - NASA Global Modeling and Assimilation Office's Goddard Earth Observing System
  • EMC GEFSv12 - Global Ensemble Forecast System v12
  • ESRL FIMr1p1 - Finite-Volume cubed-sphere Atmospheric Model
  • RSMAS CCSM4 - Community Climate System Model 4
